Bakmi Library for STM32F103

Features:
- Dual motor PWM control on fixed pins
- Servo control (PA11)
- DIP Switches for mode select (PB7, PB8, PB9)
- Start Module and Manual buttons (PB11, PB10)
- Analog input selection with threshold compare

Usage:
- Copy Bakmi folder into Arduino/libraries/
- Open Bakmi_Test.ino from examples

See README.md for more info.
